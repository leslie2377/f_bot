## 1. 프로젝트 개요

### 1.1 목적
프리텔레콤(프리티) 셀프개통 서비스 이용 고객을 위한 AI 기반 상담 챗봇을 구축한다. FAQ, 이용약관, 판매 상품(요금제) 안내를 자동으로 제공하여 고객 편의성을 높이고 상담 부하를 줄인다.

### 1.2 배경
- 프리텔레콤은 SKT/KT/LG U+ 3개 통신망 알뜰폰 서비스 제공
- 셀프개통은 약 5~10분이면 완료 가능하나, 과정 중 다양한 문의 발생
- 고객센터 운영시간(09:00~18:00, 평일)에만 상담 가능
- 웹사이트 FAQ는 동적 로딩 방식으로 탐색이 불편

### 1.3 대상 사용자
- 프리텔레콤 셀프개통을 시도하는 신규/번호이동 고객
- 요금제 비교 및 상품 정보를 원하는 잠재 고객
- 개통 후 문제(유심 인식 오류 등) 해결이 필요한 기존 고객

### 3.2 기술 스택

| 구분 | 기술 | 선택 이유 |
|------|------|-----------|
| **Frontend** | React + Vite | 가벼운 채팅 위젯, iframe/스크립트 임베드 가능 |
| **Backend** | Node.js + Express | 빠른 개발, AI API 연동 용이 |
| **AI Engine** | Claude API (Anthropic) | 한국어 성능 우수, 긴 컨텍스트 지원 |
| **데이터** | JSON 파일 기반 | 별도 DB 불필요, 관리 용이 |
| **배포** | Vercel (Frontend) + Render/Railway (Backend) | 무료 티어 활용 가능 |

| 데이터 | 출처 | 수집 방법 | 형식 |

### 4.2 수집된 핵심 정보 (현재까지)

**셀프개통 정보:**
- 신규가입: 09:00~20:00 / 번호이동: 10:00~19:00
- 준비물: 신분증(주민등록증/운전면허증), 인증서(네이버/토스/신용카드/공동인증서)
- 소요시간: 약 5~10분
- 3개 통신망(SKT/KT/LG U+) 모두 셀프개통 가능

**고객센터:**
- SKT 알뜰폰: 1661-2207
- KT 알뜰폰: 1577-4551
- U+ 알뜰폰: 1588-3615
- 민원실: 02-3489-7351

**요금제 (일부):**
- 음성기본데이터5G: 2,750원 (5GB/무제한통화/150건문자)
- 음성기본데이터10G: 100원 프로모션 (10GB/무제한/무제한)
- 더든든한500분15G: 2,750원~7,600원 (망별 상이)
- 더든든한500분20G: 8,800원~12,980원 (망별 상이)
- 하나은행/CU/신한카드 등 제휴 요금제 다수

**가입 제한:**
- 1인 최대 9회선 (선/후불 통신망별 3개)
- 번호이동: 이전 통신사 3개월 경과 필요
- 180일 내 3회선 이상 개통 제한
- 외국인: 선불/후불 각 1회선

## 5. AI 응답 전략

### 5.1 System Prompt 설계
```
당신은 프리텔레콤(프리티) 셀프개통 전문 상담사입니다.
- 친절하고 정확한 한국어로 답변합니다.
- FAQ, 약관, 요금제 데이터를 기반으로 답변합니다.
- 데이터에 없는 내용은 고객센터 연결을 안내합니다.
- 요금제 추천 시 고객의 사용 패턴을 먼저 확인합니다.
```

### 5.2 컨텍스트 주입 방식
1. 사용자 질문 수신
2. 질문 카테고리 분류 (FAQ/약관/상품/기타)
3. 관련 데이터를 컨텍스트에 포함
4. Claude API 호출하여 응답 생성
5. 응답 반환

### 5.3 답변 불가 시 폴백
- "정확한 답변을 드리기 어렵습니다."
- 고객센터 번호 + 운영시간 안내
- 1:1 문의 링크 제공 (freet.co.kr/customer/inquiry/form)
